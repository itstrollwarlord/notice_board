# Notice Board Management System

A simple web-based notice board built with PHP and MySQL. It allows administrators to manage notices through a secure dashboard while users can view published notices.

## Features

- User authentication
- Admin dashboard
- Create, edit, and delete notices
- Notice categories
- Responsive design

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
